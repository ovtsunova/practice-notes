# Цель:

### Написать приложение на flutter для заметок, где пользователь может:

- вводить в текстовое поле данные;
- при нажатии на кнопку "Сохранить" введённый пользователем текст отобразится ниже текстового поля в виде списка;
- удалять созданные заметки, последством выбора заметки и нажатия кнопки "Удалить";
- редактировать созданные заметки, посредством выбора заметки в списке, нажатия на кнопку "Редактировать".
